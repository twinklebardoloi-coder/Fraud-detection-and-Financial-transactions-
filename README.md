# Financial Fraud Detection & Risk Management

## Overview
This project focuses on detecting fraudulent financial transactions using Machine Learning techniques and visualizing insights through a Tableau dashboard. 

Fraud detection is critical for digital payment platforms because even a small percentage of fraudulent transactions can lead to significant financial loss, reputational damage, and regulatory issues. This system uses both supervised and unsupervised learning models to identify suspicious transactions and provide actionable insights.

The final results are presented through an interactive Tableau dashboard to support business decision-making and fraud monitoring.

---

## Objectives
- Detect fraudulent transactions using machine learning models
- Compare supervised and unsupervised fraud detection approaches
- Handle class imbalance using SMOTE
- Evaluate model performance using metrics such as ROC, Accuracy, Precision, Recall, and F1 Score
- Visualize fraud patterns and insights using Tableau

## Machine Learning Models

### Supervised Model
- Random Forest Classifier

Used to classify transactions as fraudulent or legitimate based on labeled data.

### Unsupervised Model
- Isolation Forest

Used for anomaly detection to identify unusual transactions that may indicate fraud.
